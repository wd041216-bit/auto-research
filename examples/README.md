## `converged-council-hybrid`

A minimal structurally valid hybrid workspace used by CI.

This example is intentionally synthetic. It demonstrates artifact shape and gate behavior, not a real scientific claim, benchmark, citation, or paper.

Run:

```bash
python3 scripts/validate_research_gates.py examples/converged-council-hybrid --mode hybrid
python3 scripts/build_review_packet.py examples/converged-council-hybrid
```
